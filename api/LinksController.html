LinksController

Controller allowing the user to follow links to locate a document on other online services.

Since finding some of these links (such as Mendeley and citeulike) require querying an external server for a piece of JSON, analyzing that JSON, and responding to it, we want to separate this out into another controller that only fetches those links when the user asks for them.

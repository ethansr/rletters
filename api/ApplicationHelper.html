add_class(name, attrs) click to toggle source

Add a class to a given element. Used to set IE-specific classes on the HTML element in ie_html. Thanks to Peter Gumeson’s HTML5 Boilerplate gem for this. 

    # File app/helpers/application_helper.rb, line 10
10:   def add_class(name, attrs)
11:     classes = attrs[:class] || ''
12:     classes.strip!
13:     classes = ' ' + classes if !classes.blank?
14:     classes = name + classes
15:     attrs.merge(:class => classes)
16:   end

ie_html(attrs={}, &block) click to toggle source

Produce an <html> element with the appropriate classes and conditional comments for detecting various versions of IE. 

    # File app/helpers/application_helper.rb, line 20
20:   def ie_html(attrs={}, &block)
21:     attrs.symbolize_keys!
22:     haml_concat("<!--[if lt IE 7]> #{ tag(:html, add_class('ie6', attrs), true) } <![endif]-->".html_safe)
23:     haml_concat("<!--[if IEMobile 7]>    #{ tag(:html, add_class('iem7', attrs), true) } <![endif]-->".html_safe)
24:     haml_concat("<!--[if IE 7]>    #{ tag(:html, add_class('ie7', attrs), true) } <![endif]-->".html_safe)
25:     haml_concat("<!--[if IE 8]>    #{ tag(:html, add_class('ie8', attrs), true) } <![endif]-->".html_safe)
26:     haml_concat("<!--[if (gte IE 9)|!(IE)]><!-->".html_safe)
27:     haml_tag :html, attrs do
28:       haml_concat("<!--<![endif]-->".html_safe)

30:     end

locale_name(locale) click to toggle source

Get the user-friendly name of a locale. Though we don’t currently have any, this does have support for dialects. (Rails I18n support for dialects is currently spotty.) 

    # File app/helpers/application_helper.rb, line 41
41:   def locale_name(locale)
42:     if locale.index('-').nil?
43:       str = I18n.t :"languages.#{locale}"
44:     else
45:       str = I18n.t :"languages.#{locale}", :default => [:"languages.#{locale.split('-')[0]}", "Unknown Language"]
46:     end
47:     str
48:   end
